# دليل المسؤول — SoaBra v1.0

## إدارة الأدوار
- مسار: **الإعدادات → إدارة الأدوار** (Owner فقط — محمي بـ RLS عبر `is_owner()`).
- الأدوار المتاحة: `owner, ciso, dpo, infra_admin, finance_admin, department_manager, project_manager, team_member, guest`.
- الإسناد عبر دالة RPC `has_role` ويتم التسجيل في `audit_events`.

## مركز التدقيق
- مسار: **الإعدادات → مركز التدقيق**.
- يعرض آخر 100 حدث، يُحدَّث كل 30 ثانية. مصدر البيانات: جدول `audit_events`.

## لوحة المحركات
- مسار: **الإعدادات → لوحة المحركات**.
- يستهلك `engine_jobs` مع Realtime — أي تحديث ينعكس فورًا.
- المهام تنتقل تلقائيًا `planned → active → completed` عبر cron كل دقيقة (`engine-jobs-worker`).

## Outbox / DLQ
- جدول `event_outbox`: أحداث في انتظار النشر. `event_dlq`: أحداث فشلت 3 محاولات.
- Edge Function `outbox-relay` تعمل كل دقيقة عبر `pg_cron`.

## خريطة التبعيات
- مسار: **الإعدادات → خريطة التبعيات**. تعرض جدول `dependencies` كرسم بياني دائري.

## سوق الأدوات
- مسار: **الإعدادات → سوق الأدوات**. تسجيل أدوات قابلة لإعادة الاستخدام داخل السبورات المركزية.

## النسخ الاحتياطي
- النسخ الاحتياطي مُدار عبر Supabase تلقائيًا. للاستعادة استخدم لوحة Supabase.
