# دليل المستخدم — SoaBra v1.0

## التنقل السريع
- **Cmd/Ctrl + K**: فتح البحث الموحّد عبر المشاريع والمهام والأدوات والسبورات والأقسام.
- **الشريط الجانبي**: التنقّل بين المشاريع، الأقسام، التخطيط، الأرشيف، والإعدادات.

## الوحدات الرئيسية
- **المشاريع**: إنشاء وإدارة المشاريع المرتبطة بقاعدة البيانات المركزية.
- **الأقسام**: لوحات الأقسام (CRM, Finance, HR, ...) مع إجراءات وتقارير.
- **التخطيط**: لوحة الكانفس التعاونية للسبورات.
- **الأرشيف**: عرض الكيانات المؤرشفة.
- **الإعدادات**: الحساب، الأمان، الأدوار، التدقيق، لوحة المحركات، خريطة التبعيات، سوق الأدوات.

## الأدوار الافتراضية
- **Owner**: تحكّم كامل (يُسنَد تلقائيًا لأول مستخدم).
- **Team Member**: تنفيذ مهام، عرض مشاريعه.
- **Project Manager / Department Manager**: حسب نطاق ABAC.

## استكشاف الأخطاء
- في حال ظهور "حدث خطأ في …" استخدم زر "إعادة المحاولة"؛ إذا تكرّر، راجع وحدة التحكّم في المتصفح.
